Learn Cloud Computing with AWS

Learn about the different tools and systems involved in cloud computing and dev ops. Explore popular cloud solutions and the benefits to cloud computing along with the core cloud architecture concepts and approaches.

You’ll also learn about the pipelines, integration techniques and workflows involved in dev ops. You’ll understand the methods and tools involved in delivery and deployment. Plus, you’ll learn about the core concepts that underpin the use of containers such as Docker and Kubernetes and how they are used. Finally, discover how all the different dev ops principles come together to form the practice of site reliability engineering.

What you'll learn

  • Learn the basics of AWS (Amazon Web Services)
  • Learn the fundamentals of AWS (EC2, ELB, and ASG)
  • Learn how to construct AWS infrastructure
  • Understand the basic cloud computing concepts
  • Develop practical skills whilst utilizing several AWS services
  • How to integrate various services together

    • Requirements

  • No prior experience is necessary in any way
  • This course is designed for absolute beginners to cloud computing and AWS
  • No IT prerequisites are required

    • Join Edutech for this course if you want to master the basics of Cloud computing on AWS and kick-start your career in cloud computing.

    Who this course is for:

    • Those who have an interest and desire to learn cloud computing
    • Beginners learning about cloud computing for the first time
    • Individuals who want to learn about Amazon Web Services (AWS)
    • Those that want to add to their pre-existing cloud knowledge
    • Programmers/developers with a keen interest in learning about the foundations of cloud development
